Tutorial

It consists of the following inter-related modules:

  1. What-next recommender
  2. Journal
  3. Learning + Affirmations + Quiz
  4. Mindfulness
  5. Tasks
  6. Time planning

The app tries to be as intuitive as possible.

If You need more information, please press the help icon on given page or dialog.

There You also find a button for quick targeted feedback. Where it's enough to write for example something really short like "make the buttons here bigger" and we will know to which page or dialog it refers.

It is optional (but recommended, as we might also provide hints or ask clarifying questions) to give contact information for us to be able to reply to You.

Your feedback helps us a lot - thank You. And we try to apply it as much as possible.

Certain User Interface items can be long-pressed for an alternative action.
